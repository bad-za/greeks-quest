# Greeks Quest — интерактивный курс по крипто-опционам

Обучающая игра: 12 уровней от «что такое колл» до волатильностных стратегий и
исторических сценариев (крах FTX, одобрение ETF, халвинг). Под капотом — честный
Блэк-Шоулз: все премии, греки и P&L считаются, а не нарисованы.

## Запуск

```bash
npm install
npm run dev
```

Открой http://localhost:5173

## Структура

- Акт I — основы: коллы, путы, инверсные опционы Deribit
- Акт II — греки: дельта, гамма, тета, вега, IV
- Акт III — стратегии: спреды, covered call, стрэддлы, кондоры
- Акт IV — улыбка/скью, риск-менеджмент, финальный экзамен

Прогресс, XP, виртуальный портфель $10,000 и ачивки хранятся в localStorage.
Сбросить — кнопка внизу карты курса.

## Telegram Mini App

Приложение интегрировано с Telegram WebApp SDK: нативная кнопка «Назад»,
хаптика на квизах и миссиях, синхронизация прогресса через CloudStorage
(работает между устройствами). В обычном браузере всё это тихо отключается.

Чтобы запустить как Mini App:

1. **Собери статику**: `npm run build` → папка `dist/`.
2. **Выложи `dist/` на любой HTTPS-хостинг** (GitHub Pages, Cloudflare Pages,
   Vercel, Netlify — все бесплатны; пути в сборке относительные, работает откуда угодно).
3. **Создай бота**: в [@BotFather](https://t.me/BotFather) → `/newbot`, придумай имя.
4. **Привяжи Mini App**: `/newapp` в BotFather → выбери бота → укажи URL хостинга,
   название и иконку. Или проще: `/mybots` → бот → Bot Settings → Menu Button →
   укажи URL — приложение откроется по кнопке в чате с ботом.
5. Для быстрой проверки без хостинга: `npm run dev` + HTTPS-туннель
   (`npx localtunnel --port 5173` или ngrok) и временно укажи URL туннеля в BotFather.

Учебный материал, не финансовый совет.
